# Teorie her

Proč hry fungují a jak o nich přemýšlet: design mechanik, psychologie hráče, proces tvorby — a taky mentální stránka řemesla, protože samouk bojuje stejně často s motivací jako s technologií.

Kapitoly:

- **[Začátky bez zkušeností](zacatky-bez-zkusenosti.md)** — jak vůbec začít dělat hry, když neumíš „nic": malý rozsah, sdílení práce, motivace vs. vůle.
- **[Smyčky a řetězce](smycky-a-retezce.md)** — dva základní tvary herního designu: co smyčka vysvětluje, co už ne, a proč řetězce nesou emoce.
