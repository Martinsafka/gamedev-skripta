# Zápisky

Lekce z vlastních projektů — deníky vývoje a shrnutí pracovních debat, destilované do čitelné podoby. Všechno anonymizované: **identity nejsou pro myšlenku důležité.**

Zápisky se do skript teprve stěhují. Až tu budou, každý ponese strukturu: kontext → co se stalo → co si z toho beru → souvislosti s kapitolami teorie a praxe.
